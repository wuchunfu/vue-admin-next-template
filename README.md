#### 🌈 介绍

基于 vue3.x + CompositionAPI + typescript + vite + element plus + vue-router-next + next.vuex，适配手机、平板、pc 的后台开源免费模板，希望减少工作量，帮助大家实现快速开发。

#### 💒 代码仓库

- vue3.x 版本 <a href="https://github.com/wuchunfu/vue-admin-next-template" target="_blank">github.com/wuchunfu/vue-admin-next-template</a>

#### 🚧 安装 cnpm, yarn, pnpm

- 复制代码(桌面 cmd 运行) 
  ```bash
  cnpm install -g cnpm --registry=https://registry.npmmirror.com
  ```
- 复制代码(桌面 cmd 运行) 
  ```bash
  npm install -g yarn
  ```
- 复制代码(桌面 cmd 运行)
  ```bash
  npm install -g pnpm
  ```

#### ⚡ 使用说明

建议使用 pnpm，因为 yarn 有时会报错。<a href="http://nodejs.cn/" target="_blank">node 版本 > 12xx.xx.x</a>

```bash
# 克隆项目
git clone https://github.com/wuchunfu/vue-admin-next-template.git

# 进入项目
cd vue-admin-next-template

# 安装依赖
pnpm install

# 运行项目
pnpm run dev

# 打包发布
pnpm run build
```
